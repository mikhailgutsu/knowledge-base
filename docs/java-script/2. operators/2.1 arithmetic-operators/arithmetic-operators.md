# 1. Arithmetic Operators

Used for mathematical operations.

- `+` addition;
- `-` subtraction;
- `*` multiplication;
- `/` division;
- `%` modulus (remainder);
- `**` exponentiation (power);
- `++` increment;
- `--` decrement.

```js
let a = 10;
let b = 3;

console.log(a + b); // 13
console.log(a - b); // 7
console.log(a * b); // 30
console.log(a / b); // 3.333...
console.log(a % b); // 1  (remainder of 10/3)
console.log(a ** b); // 1000 (10^3)

let x = 5;
x++;
console.log(x); // 6
x--;
console.log(x); // 5
```
